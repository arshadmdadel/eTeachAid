# eTeachAid – JavaFX Tuition Platform

## Overview
**eTeachAid** is an AOP-based tuition management platform developed using **JavaFX**. The platform connects **students** and **teachers/tutors** in an organized and user-friendly way. Students can search for tutors, send tuition requests, and after a tutor accepts the request, both users can interact through the platform.

The project also includes a complete **database management system** for storing user information, tutor requests, profiles, and tuition details.

---

## Features

- Student & Tutor Registration/Login
- Tutor Search System
- Tuition Request & Acceptance System
- Student Profile and Bio Management
- Tutor Can View Accepted Student Details
- Database Connectivity using JDBC
- User-Friendly JavaFX Interface
- Secure User Data Management

---

## Technologies Used

- **Java**
- **JavaFX**
- **FXML**
- **Scene Builder**

---

## System Workflow

1. Student creates an account  
2. Tutor creates an account  
3. Student searches for tutors and sends requests  
4. Tutor accepts the request  
5. After acceptance, the tutor can view:
   - Student details
   - Student bio
   - Contact information  
6. All information is stored and managed using a database system

---

## Database Features

The database system is used to:

- Store student and tutor information
- Manage tuition requests
- Save accepted tutor-student relationships
- Handle user authentication

---

## Project Goal

The main goal of **eTeachAid** is to simplify the process of finding tutors and managing communication between students and teachers through a digital platform.
